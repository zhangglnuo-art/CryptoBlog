「我转的币怎么还没到？」「这个地址收到钱了吗？」「这个新代币的合约靠不靠谱？」——这些问题的答案，全都写在**区块浏览器**上。学会用它，你就有了看穿链上一切的「透视镜」。

本文以最常用的 **Etherscan**（以太坊区块浏览器）为例，讲清楚怎么查。

## 区块浏览器是什么

[区块链是公开透明的账本](/blog/what-is-blockchain)，每一笔交易、每个地址的余额和历史都公开可查。**区块浏览器就是查询这个公开账本的网站**。以太坊用 Etherscan，其他链有各自对应的浏览器（下文有列举）。

它不需要登录、不掌握你的资产，只是一个「只读」的查询工具，任何人都能用。

## 用交易哈希查一笔交易

每笔链上交易都有一个唯一的「交易哈希（Transaction Hash / TxID）」，一长串以 `0x` 开头的字符。你在钱包或[交易所的提现记录](/blog/exchange-withdraw-to-wallet)里能找到它。

把交易哈希粘到 Etherscan 搜索框，就能看到：

- **Status（状态）**：Success 成功 / Pending 处理中 / Failed 失败
- **确认数**：确认越多越稳妥
- **From / To**：转出和转入地址
- **数量与 [Gas 费](/blog/gas-fee-explained)**：这笔转了多少、花了多少手续费
- **时间戳**：交易发生的时间

**转账没到账时**，第一步就是拿交易哈希来这里查：如果显示 Success，那多半是钱包没添加对应代币或没切到对应网络，而不是币丢了。

## 查一个钱包地址

把任意 `0x` 地址粘进搜索框，可以看到这个地址的：

- **余额**：持有多少 ETH
- **持有的代币**：有哪些 ERC-20 代币和数量
- **交易历史**：所有转入转出记录

这对核对「对方到底收到没有」、或检查自己地址的活动很有用。注意：地址是公开的，查地址不等于能动它的钱——**能动钱的只有[私钥/助记词](/blog/crypto-wallet-security)的持有者**。

## 查代币与合约（防踩雷）

买一个没听过的新代币前，用区块浏览器做点基础核查，能避开不少[骗局](/blog/common-crypto-scams)：

- **合约是否开源验证**：Etherscan 上标了 "Contract Source Code Verified" 的更透明
- **持有人分布（Holders）**：如果绝大多数代币集中在极少数地址手里，砸盘/操纵风险高
- **交易活动**：是否有正常的买卖，还是只能买不能卖（[貔貅盘](/blog/common-crypto-scams)信号）

区块浏览器不能保证一个项目一定安全，但能帮你排除掉一批明显可疑的。

## 其他常见区块浏览器

不同[公链](/blog/public-chains-compared)有各自的浏览器：

- 以太坊 → Etherscan
- BNB Chain → BscScan
- Tron 波场 → Tronscan
- Solana → Solscan / Solana Explorer
- Arbitrum、Base 等 Layer 2 → 各自对应的 Scan

用法都大同小异：搜交易哈希、搜地址、搜代币合约。

## 常见问题

**Q：查交易需要付费或登录吗？**
不需要。区块浏览器是免费的公开只读工具，不用注册、不用连钱包。

**Q：交易一直是 Pending 怎么办？**
通常是 Gas 给太低或网络拥堵。可等待，或在钱包里用「加速 / 替换」功能提高 Gas 重新广播（视钱包而定）。

**Q：在区块浏览器连接钱包安全吗？**
查询本身不需要连钱包。它也提供一些需要连钱包的高级功能，但和任何 DApp 一样，**连接和签名前都要看清在授权什么**。

## 小结

区块浏览器是每个加密用户都该会用的基本功：用交易哈希查状态、用地址查余额与历史、用合约地址做上币前的基础核查。转账没到账先别慌，来这里一查便知。掌握了它，链上世界对你就不再是黑箱，而是一本随时可查的公开账本。
