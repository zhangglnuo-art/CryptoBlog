NFT 大概是加密世界里最容易被误解的概念：有人把它等同于「天价猴子头像」，有人觉得「右键就能保存的图片凭什么值钱」。

抛开炒作，NFT 本身其实是个不复杂的技术概念。这篇文章讲清楚它到底是什么、你买到的究竟是什么，以及热潮退去后剩下了什么。

## 从「同质化」说起

NFT 全称 **Non-Fungible Token，非同质化代币**。要理解它，先理解什么叫「同质化」。

- **同质化**：你手里的 1 个 USDT 和我手里的 1 个 USDT **完全等价、可以互换**，谁也不吃亏。像人民币、像[比特币](/blog/bitcoin-vs-ethereum)，都是同质化的。
- **非同质化**：每一个都**独一无二、不能随便互换**。像演唱会门票（座位不同）、像房产证（地址不同）、像球星签名球衣。

所以 NFT 的定义可以简化成一句话：**链上独一无二、可以验证归属的凭证。**

USDT 记的是「某地址有多少个」，NFT 记的是「某地址拥有哪一个」。

## 它和普通代币的区别

| 对比项 | 普通代币（如 [USDT](/blog/stablecoin-usdt-usdc)） | NFT |
| --- | --- | --- |
| 是否可互换 | 可以，1 个换 1 个 | 不可以，每个都不同 |
| 有无编号 | 无 | **有唯一编号（Token ID）** |
| 能否拆分 | 能（0.5 个 USDT） | 通常不能 |
| 记录什么 | 你有**多少** | 你拥有**哪一个** |
| 典型用途 | 支付、计价 | 所有权、身份、门票 |

两者的底层其实是同一套东西：都是链上的[智能合约](/blog/what-is-smart-contract)在维护一张账本。区别只是账本记法不同——一个记数量，一个记编号归属。

## 关键认知：你买 NFT 时买到的是什么

这是最多人搞错、也最该说清楚的一点。

**买一个图片类 NFT，你买到的通常不是那张图片本身，而是链上一条「你拥有这个编号」的记录。**

原因很现实：图片文件太大，直接存链上成本高得离谱。所以绝大多数 NFT 的做法是——**链上只存一个指向图片的链接**，图片本身放在别处（去中心化存储或普通服务器）。

由此带来几个必须知道的后果：

- **「右键保存」和你的所有权不冲突**：别人复制图片，就像你复制一张名画的照片——原作的归属记录没变。这是个常被拿来嘲讽的点，但它没抓住重点。
- **图片可能会消失**：如果图片存在项目方的普通服务器上，服务器一关，你的 NFT 可能就只剩一个指向 404 的链接。存在去中心化存储上的更可靠。
- **版权 ≠ 所有权**：买了 NFT 不等于买了这个形象的商业版权。**具体权利完全取决于项目方的条款**，很多项目其实什么版权都没给你。

买之前先问清楚：链上存的是什么、图片存在哪、我到底获得了什么权利。

## 除了头像，NFT 还能干什么

热潮期大家只看到炒作，但「链上唯一凭证」这个能力本身有不少务实的用途：

- **门票与凭证**：演出票、活动通行证，可验真伪、可追溯转手
- **域名**：如以太坊域名（把一长串 `0x` 地址映射成一个好记的名字）
- **游戏道具**：装备、皮肤归玩家所有，能跨市场自由交易
- **会员资格与身份**：持有某个 NFT 即可进入社群、享受权益
- **链上凭证**：学历、证书、参与证明等不可转让的记录

值得注意的是，前面讲过的 [DeFi](/blog/what-is-defi) 里也有 NFT 的身影——一些协议用 NFT 来表示你的[流动性仓位](/blog/impermanent-loss-explained)，因为每个仓位的价格区间、份额都不一样，天然就是「非同质化」的。这类用法和炒图片毫无关系。

## 风险与常见骗局

NFT 领域的坑比一般代币更多，因为它的定价更主观、更依赖叙事：

- **流动性极差**：代币随时能按市价卖出，NFT **必须有人愿意接盘**才卖得掉。热度一过可能挂几个月无人问津，账面价格毫无意义。
- **洗售交易（Wash Trading）**：自己左手倒右手抬高成交价，制造「地板价很高」的假象诱人接盘。
- **假冒系列**：任何人都能铸造一个和知名项目一模一样的 NFT。**认合约地址，别认图片和名字**——这点和[假代币](/blog/what-is-smart-contract)是一个道理。
- **恶意授权**：假 mint 网站、假空投诱导你签名，实则是转走资产的[授权](/blog/token-approval-and-revoke)。NFT 领域这类[钓鱼](/blog/common-crypto-scams)尤其猖獗。
- **项目方跑路**：卖完就解散、路线图全成空话，是常态而非例外。

安全习惯和链上其他操作一致：**只从官方渠道进入、签名前看清在授权什么、大额资产放[冷钱包](/blog/cold-vs-hot-wallet)。**

## 常见问题

**Q：右键就能保存图片，NFT 凭什么值钱？**
你保存的是图片副本，链上「谁拥有」的记录没变，就像临摹名画不影响原作归属。至于**值不值那个价**，那是市场定价问题——技术上「唯一归属」是成立的，但成立不等于值钱。

**Q：NFT 现在还有价值吗？**
炒作降温后，绝大多数图片类项目价格大幅回落，这是事实。但「链上唯一凭证」作为一项技术仍在门票、域名、游戏、身份等场景里被使用。**把技术和当时的价格泡沫分开看**更客观。

**Q：买 NFT 需要什么？**
一个支持该链的钱包（如 [MetaMask](/blog/metamask-guide)）、链上的原生代币付款和 [Gas 费](/blog/gas-fee-explained)，然后在交易市场购买。不同链有各自的 NFT 生态。

**Q：NFT 能像币一样随时卖掉吗？**
不能。这是它和代币最大的实际差别——**成交需要有人恰好想买你这一个**。买之前务必想清楚流动性风险。

**Q：我的 NFT 存在哪？会不会丢？**
所有权记录在链上，只要[私钥](/blog/crypto-wallet-security)安全就不会丢。但它指向的**图片文件**可能因存储服务下线而失效——这两件事是分开的。

## 小结

NFT 的本质不神秘：**链上独一无二、可验证归属的凭证**。它和 USDT 这类代币的区别，只是「记数量」和「记编号」之别。买图片类 NFT 时，你获得的是一条所有权记录，而非图片本身，权利范围完全看项目条款。抛开天价头像的喧嚣，这项技术在门票、域名、游戏道具、身份凭证上仍有扎实用途。真要参与，记住两件事：**流动性远比代币差，且假冒与钓鱼格外多。**
